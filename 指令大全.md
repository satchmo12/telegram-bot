# 机器人指令大全（速查版）

## 怎么用
- 支持两种写法：`指令` 或 `/指令`（例如：`签到`、`/签到` 都可以）。
- 有参数时用空格分开：如 `种植 小麦 全部`。
- 多数管理指令需要管理员权限，且群功能已开启。

## 一、最常用指令（先记这 20 条）
- `功能介绍`：查看功能简介
- `签到`：每日签到
- `我的金币`：看余额
- `用户信息`：看个人信息
- `财富排行`：看金币排行
- `存款` / `取款`：银行存取
- `我的农场`：查看农场
- `种植 <作物> [地块/全部]`
- `农场收获` / `偷菜`
- `我的花园` / `种花 <花名>`
- `我的牧场` / `饲养 <动物>`
- `我的库存` / `出售 <物品> <数量>`
- `商店` / `采购 <物品> <数量>`
- `真心话` / `大冒险`
- `成语接龙` / `提示` / `结束接龙`
- `五子棋` / `加入五子棋` / `下棋 <位置>`
- `抢红包` / `发红包 <金额> <个数>`
- `查询 <币名>`（查币价）
- `翻译 <文本>`
- `群状态`（管理员常用）

## 三、参数模板（最容易输错的）
- `公司注册 [类型] [公司名称]`
